# ServiceCart
# Service Cart Web Application

## Project Overview

Service Cart is a web-based platform that connects customers with local service providers such as plumbers, electricians, cleaners, and more. The system allows users to register, browse services, and book services easily.

* User Registration & Login (Customer / Service Provider / Admin)
* Service Listing Page
* Search Services Functionality
* Service Booking System
* Service Provider Dashboard
* Role-Based Access Control

## Technologies Used

### Frontend:

* HTML
* CSS
* Bootstrap
* JavaScript

### Backend:

* PHP

### Database:

* MySQL

## Database Structure

### Users Table:

* id
* name
* email
* password
* role (customer / provider / admin)
* status (pending / approved)

## Conclusion

This project demonstrates a complete service booking system using PHP and MySQL with role-based authentication and real-world service booking features.
